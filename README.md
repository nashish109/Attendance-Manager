# Acadmetrix Portal  

A professional academic attendance management system designed to help students monitor, analyze, and maintain their attendance performance with structured insights.

Welcome to **Acadmetrix Portal**:  
Live Application: https://acadmetrix-portal.vercel.app  

Acadmetrix transforms basic attendance tracking into a performance-focused academic monitoring system. It provides students with clear visibility into their overall attendance, subject-wise performance, and compliance status with institutional thresholds.

Through a clean and structured interface, the platform offers:

- A comprehensive dashboard displaying overall attendance and compliance status  
- Subject-wise analytics with deviation tracking and risk indicators  
- Attendance trend visualization for performance consistency  
- Projection insights to calculate allowable absences and recovery requirements  

Built with a modern web stack and deployed on Vercel, Acadmetrix ensures a responsive, reliable, and professional user experience across devices.
